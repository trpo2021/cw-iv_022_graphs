Здравствуйте, данное приложение создано для расчета кратчайшего,
    длиннейшего и всевозможных путей между двумя городами с помощью матрицы
        смежности.Эта программа работает на ориентированном графе без петель.

    Для того,
    чтобы приложение начало свою работу,
    нужно запустить его с ключом '-m' и с именем файла с Вашей
        матрицей.После этого выберите,
    что вы хотели бы узнать
    : '-n' -
      кол - во маршрутов между городами '-s' -
      кратчайший путь между городами '-l' -
      длиннейший путь между городами Вводится один из предложенных ключей
          .Если вы не ввели какой либо ключ после названия файла с матрицей,
    то наша программа снова выведет инструкцию,
    чтобы вы увидели какие ключи вы забыли
                    .

                Примеры запуска
    :./ bin / graph -
            m matrix4.txt - n./ bin / graph - m matrix4.txt -
            s

                Заполните файл с матрицей по шаблону
    : ++++++++++++++++++++++++++++++++++4 <
        -количество городов 3 < -начальный город 1 < -конечный город 1 - 2;
1 - 3;
1 - 4;
2 - 1;
2 - 4;
3 - 2;
3 - 4;
4 - 1;
4 - 2;
4 - 3;
++++++++++++++++++++++++++++++++++

                                Файлы с матрицей должны храниться в папке bin /
        .

        Приятного использования !
