Здравствуйте, данное приложение создано для расчета кратчайшего пути между двумя городами с помощью матрицы смежности.
Эта программа работает на ориентированном графе без петель.
Для того, чтобы приложение начало свою работу, нужно запустить его с ключом "-m" и с именем файла с Вашей матрицей. 
Заполните файл с матрицей по шаблону:

4 <- количество городов
3 <- начальный город
1 <- конечный город
ребро 1 - 1: 0
ребро 1 - 2: 1
ребро 1 - 3: 1
ребро 1 - 4: 1
ребро 2 - 1: 1
ребро 2 - 2: 0
ребро 2 - 3: 0
ребро 2 - 4: 1
ребро 3 - 1: 0
ребро 3 - 2: 1
ребро 3 - 3: 0
ребро 3 - 4: 1
ребро 4 - 1: 1
ребро 4 - 2: 1
ребро 4 - 3: 1
ребро 4 - 4: 0 

Помните, максимальное количество городов - 10.
Файлы с матрицей должны храниться в папке bin/.
Пример запуска:
./bin/graph -m matrix4.txt

Приятного использования!
